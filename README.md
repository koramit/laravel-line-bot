# Laravel LINE bot

## Installation
`composer require koramit/laravel-line-bot`

`php artisan vendor:publish --provider="Koramit\LaravelLINEBot\LINEBotServiceProvider"`

`php artisan migrate`

```aiignore
# config .env file
LINE_BOT_BASIC_ID=
LINE_BOT_CHANNEL_ID=
LINE_BOT_CHANNEL_SECRET=
LINE_BOT_CHANNEL_ACCESS_TOKEN=
```
